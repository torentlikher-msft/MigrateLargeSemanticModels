  This code is intended as a sample/accelerator to assist with migrating Large Format semantic models
  across capacities and regions in Microsoft Fabric. It performs tenant-wide scanning and workspace-level
  operations including creating workspaces, reassigning capacities, and deploying semantic models. These
  actions can have significant impact on your environment.
  
  Before using this code:
   - Thoroughly review and understand every cell before executing.
   - Test in a non-production environment first.
   - Ensure you have appropriate administrator permissions and organizational approval.
   - Back up any critical semantic models and workspace configurations.
   - Consider hard-coding a single workspace id for testing purposes.
